# tudoor Backend

## Installation

```sh
git clone https://github.com/DonIsaac/Terp-HackDay # Clone the repo
cd Terp-HackDay/Backend
npm install # Installs project dependencies
touch .env # Will store sensitive config data
# .env needs at least the following variables:
# DB_URI=<your database URI>
# SECRET=<secret key for server session>
```
